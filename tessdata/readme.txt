This software allows you to translate any text on screen. Basically it is a combination of screen capture, OCR and translation tools.

Guide how to use: https://tangaria.com/translation/

in short: copy the file 'tangaria_font.traineddata' and 'tangaria_eng.traineddata' to

Windows:
C:/Users/user-name/AppData/Roaming/Gres/ScreenTranslator/assets/tessdata

Linux:
/home/user-name/.local/share/Gres/ScreenTranslator/assets/tessdata
